Install RTL file and make .mcs file
===================================

For ddr200t:

make install BOARD=ddr200t

make mcs BOARD=ddr200t

================================

For mcu200t:

make install BOARD=mcu200t

make mcs BOARD=mcu200t

================================

For Genesys2:

make install BOARD=genesys2

make mcs BOARD=genesys2
